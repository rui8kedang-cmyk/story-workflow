# 预设配置存放目录

不同故事项目的预设配置文件存放在这里，用文件名区分。

## 命名规则

```
preset_<故事名>.json
```

示例：
- `preset_艾迪星.json` — 大发明家艾迪·星
- `preset_土松嘟嘟.json` — 土松嘟嘟

## 使用方式

1. 在工作流工具中填写"预设文件名"（如：艾迪星）
2. 点击「💾 导出预设配置」→ 自动生成 `preset_艾迪星.json`
3. 将文件放到此目录并 push 到 GitHub
4. 需要时通过「📂 导入预设配置」加载

> ⚠️ 注意：预设文件中包含 API Key，请确保仓库为私有仓库。
