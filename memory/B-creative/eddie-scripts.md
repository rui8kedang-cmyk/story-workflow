# 《大发明家艾迪·星》 - 剧本库

## 创作规范

- **时长：** 每集 45秒 ~ 1分30秒
- **结构：** 难题 → 发明 → 过程 → 结局（弄巧成拙的搞笑收场）
- **特点：** 发明装置用日常废旧物品组装，名字要有趣

### 禁忌/注意事项

- ❌ **不出现机械狗**（机械狗是独立角色线，不在单集剧本中出现）
- ❌ **避免作弊等非正能量主题**（除非故事特别搞笑才可例外）
- ✅ 难题应来自日常生活、学校、家庭中的正常烦恼

## 已完成剧本（已采纳并生成动画）

### EP. 《光合作用台灯》 ✅

**难题：** 艾迪负责照顾班级的盆栽植物，但他总是忘记把它们搬到太阳底下，叶子都黄了。

**发明：** "追光者"。用饼干盒和轮子组装而成，将花盆用胶带粘在上面。侧面有光线传感器，可以让植物自己去寻找阳光。

**过程：** 装置启动，追光者载着花盆，向窗边阳光最充足的地方移动，互相挤来挤去争抢最亮的位置。嘈杂的声音引来了校长来教室巡视，校长的光头在灯光的照耀下闪闪发光，被装置"盯上"，追着校长跑去。

**结局：** 传感器把校长那颗闪亮的光头误判为最亮的位置。"追光者"们疯狂地冲向校长，校长逃出了教室，向走廊的远处逃跑。

**笑点：** 校长光头 = 最强光源，植物追人

### EP. 《蚊子终结者》 ✅

**难题：** 夏天晚上有一只蚊子在艾迪耳边嗡嗡叫，怎么打也打不到，搞得他睡不着。

**发明：** "蚊子终结者"。一个可以高速旋转的底盘，上面是一个从玩具拆下来的激光自动瞄准系统，另一端是一个机械手臂，上面绑着苍蝇拍。

**过程：** 瞄准系统锁定蚊子，红色的激光点在黑暗的房间里疯狂移动。

**结局：** 蚊子停在了艾迪的鼻子上，艾迪大感不妙，自动系统毫不留情地触发，"啪"的一声巨响，艾迪的脸上被打出了一个大苍蝇拍的印子。

**笑点：** 蚊子停在脸上 → 自动系统拍脸，经典自伤

## 剧本模式总结

从已有剧本提炼的创作规律：

1. **难题来源：** 日常小事（照顾植物、被蚊子烦）
2. **发明命名：** 有趣的名字（"追光者""蚊子终结者"）
3. **材料来源：** 日常废旧物品（饼干盒、玩具零件、胶带、轮子）
4. **过程：** 一开始看起来要成功了...
5. **转折：** 意想不到的变量出现（校长光头、蚊子停脸上）
6. **结局：** 弄巧成拙，视觉冲击力强的搞笑画面
7. **受害者：** 可以是艾迪自己，也可以是旁人（校长等）
