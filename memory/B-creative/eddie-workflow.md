# 《大发明家艾迪·星》 - 创作工作流

## 流程

1. **批量生成：** 每次生成 10 个剧本
2. **用户筛选：** 从中挑选可用的
3. **反馈分析：** 分析被采纳的原因，提炼规律，提高下一轮采纳率
4. **修改迭代：** 如果故事不错但需要调整，根据修改意见完善
5. **正式收录：** 最终可用的剧本收录到 eddie-scripts.md
6. **分镜输出：** 从收录剧本中选出，生成 Seedance 2.0 视频用的分镜脚本

## 质量演进

每轮结束后更新"采纳分析"，让后续生成越来越精准。

## 工具链

- 剧本创作 → 本流程
- 视频生成 → Seedance 2.0（分镜脚本格式待确认）
