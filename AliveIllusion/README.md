## 📊 项目概览

### 核心概念
- **类型：** 生存卡牌策略游戏  
- **主题：** AI监控下的密闭空间生存  
- **核心循环：** 衰减 → 事件 → 摸牌 → 出牌 → 结算 → 死亡  
- **独特卖点：** 等轴场景背景 + 写实卡牌 + 冷色调数据面板 UI  

### 数值设计 (v7定稿)
- **平均生存：** 16.5天 (3000局模拟验证)  
- **核心参数：** 摸4打3，牌库14张，衰减饱食-12/天、水分-18/天  
- **死因分布：** 精神81.6% | 水分17.2% | 饱食1.2%
