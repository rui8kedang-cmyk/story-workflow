# 小故事创作工作流 (story-workflow)

> GitHub: https://github.com/rui8kedang-cmyk/story-workflow
> 更新：2026-04-06

## 项目简介

基于 AI 的小故事批量创作工具，一个单页 HTML 应用（纯前端，无后端），用于《大发明家艾迪·星》系列短视频的剧本 → 分镜 → Seedance 2.0 提示词全流程生产。

## 文件结构

```
story-workflow/
├── index.html            # 主工作流页面（桌面端，1770行）
├── preference-tool.html  # 偏好分析工具页面
├── presets/              # 预设配置目录
│   └── README.md
└── PROJECT.md            # 本文件
```

## 功能模块

### index.html — 主工作流
- **三面板布局**：💡头脑风暴 → 📝剧本创作 → ✨分镜提示词
- **管道图（Pipeline）**：可视化 5 步流程（创意→筛选→剧本→分镜→提示词）
- **AI 集成**：调用 LLM API（可配置模型/API Key/端点）
- **批量生成**：每次生成 10 个创意，用户筛选后进入下一步
- **分镜表格**：自动生成镜号、画面描述、角色、台词、提示词
- **日志面板**：运行时 API 调用日志，方便排查问题
- **JSON 自动修复**：处理 AI 返回的不合规 JSON（未加引号中文值、尾逗号等）
- **预设导入/导出**：配置参数可保存为预设文件
- **侧栏配置**：API 设置、生成参数等

### preference-tool.html — 偏好分析
- 上传已有故事/技能文件
- AI 分析偏好风格
- 用于提高后续生成质量

## 技术栈

- 纯 HTML + CSS + JavaScript，无框架依赖
- 暗色主题 UI（科技风）
- CSS 变量系统管理配色
- 单文件架构（所有逻辑在 HTML 内）
